# CRUD de Usuários em Java

Sistema de cadastro de usuários via terminal feito em Java.

## Funcionalidades
- Cadastrar usuário
- Listar usuários
- Buscar usuário (sem diferenciar maiúsculas/minúsculas)
- Remover usuário com confirmação
- Editar usuário

## Tecnologias
- Java 17+
- ArrayList

## Como rodar
1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Rode o arquivo `src/Main.java`
